# Timer
Desktop timer application built using Windows Forms and .NET Framework 4.7.2.

<img src="https://user-images.githubusercontent.com/31261595/157704529-8f848199-8a17-4922-8b52-a1d84a042540.png" width="400" height="300">

## Usage
Simply specify the time you want to start countdown from and hit enter on any NumericUpDown or click Start / Resume button.

<img src="https://user-images.githubusercontent.com/31261595/157706401-934406d3-e83d-4b45-b87e-469950fed05c.png" width="400" height="300">

If you want to pause, hit the Pause button.

<img src="https://user-images.githubusercontent.com/31261595/157706616-de07d04e-285b-42e8-a6e4-8ee01bc76a39.png" width="400" height="300">

If you wish to resume, hit the Start / Resume button.
When the time elapses, a MessageBox with notification will pop up. Also you will be notified with an alarm.

<img src="https://user-images.githubusercontent.com/31261595/157707097-44bfa7bd-e26e-47d6-88c7-329cbbcf2993.png" width="400" height="300">

To shut down the alarm, press OK on the MessageBox.
